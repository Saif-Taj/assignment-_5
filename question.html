    Question-3: What is event delegation in the DOM?

    ..........................................................................................................................................................................................................................................................................................................................................................................................................................

    Event delegation is a technique in JavaScript where instead of adding event listeners to multiple child elements, you add a single event listener to a parent element and use event bubbling to handle events efficiently.

    Question-4: What is event bubbling in the DOM?

    ..........................................................................................................................................................................................................................................................................................................................................................................................................................

    Event bubbling is a mechanism in the DOM where an event starts from the target element and propagates up through its parent elements.
